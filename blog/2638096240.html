nginx实验七

本文最后更新于：2023年10月25日 下午

Linux安装Redis

（1）使用wget命令下载redis-6.0.8.tar.gz

wget http://download.redis.io/releases/redis-6.0.8.tar.gz

（2）解压源码并编译安装
（3）启动 redis 服务
（4）打开另一终端，使用 redis-cli 和 redis 服务交互

PHP 使用 Redis

（1）PHP安装redis扩展
（2）修改php.ini文件

vi /usr/local/php/lib/php.ini

（3）安装完成后重启php-fpm,查看phpinfo信息，就能看到redis扩展。
